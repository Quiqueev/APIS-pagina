Instrucciones para interactuar con el bot

Para que el bot te conteste, tu tweet debe contener lo siguiente:

  1. Se debe mencionar al bot con @CovidNuevoLeon
  2. Se debe mencionar uno de los siguientes comandos: casos, fallecimientos o defunciones
  3. Se debe mencionar si se desea saber los datos totales o nuevos
  4. Se debe mencionar la fecha que se desea en el siguiente formato: Año/Mes/Día

Ejemplos

  1. @CovidNuevoLeon defunciones totales 2021/11/22
  2. @CovidNuevoLeon casos totales 2021/11/22
  3. @CovidNuevoLeon casos nuevos 2021/11/22
  4. @CovidNuevoLeon fallecimientos nuevos 2021/11/22

Consideraciones importantes

  1. El comando debe estar en minúsculas
  2. El bot regresa las fechas del día anterior si todavía no existen cifras para el día solicitado
